Here is a list of commands you should try :

  • - [list] or [ls]: lists all files in the directory.
  • - [cd] or [cd ./] in front of a file to go there.
  • - [~/] or [cd ~/] to go home.
  • - [./ example ] simply type the name of an executable to run it.
  • - [clear] or [cl]: clears all previous commands.
  • - [cat]: can be used in front of .txt files.
  • - [neofetch]: retrieves page presentation.
