A big project I have on the side is an real-time RPG game using the [Godot Engine] and [GDScript]. The combats would happen on a grid-based map and the player could chose between four classes: Healer, Tank, Mage or Melee.

For now the code is private and I do not wish to share it until the game is done and polished. Once it is, I was thinking about releasing it either for free or else it will be extremely affordable. And when it is released, the code will definitely be open source.
