This is an single page application rpg map [Git repository]. It was my first time using a JavaScript framework [Vue.js] and was made to help our class in remembering the progression of the story in our custom d&d game.The site allows users to explore an interactive RPG-style map, with customizable markers and routes for easier navigation.

Even though my understanding of component based code was not optimal at the time it still acts as a nice memento to my time spent with my colleagues and the adventures we had.
